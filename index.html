Download Options

Download ZIP

  1. Lejupielādējiet .zip arhīvu no GitHub vai šīs mājaslapas
  2. Ekstraktējiet failu uz jums piemērotāko vietu
  3. Atveriet terminālu un ar cd "ceļš uz failu" komandas palīdzību atveriet izvilkto failu
  4. Izpildiet komandu: java -jar target/aSHOPdfok.jar lai palaist programmu
Download ZIP

Git Clone

  1. Lejupielādējiet Git no git-scm.com ja tas nav ielādēts
  2. Atveriet terminālu un izpildiet komandu: git clone https://github.com/23DP1ALips/aSHOPdfok.git
  3. Pārvietojaties uz projektu ar komandu: cd aSHOPdfok
  4. Izpildiet komandu: java -jar target/aSHOPdfok.jar lai palaist programmu
